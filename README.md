# transfert-cdg-paris.fr

Site vitrine **ASC Driver** — Chauffeur privé VTC haut de gamme à Paris, spécialiste du
transfert vers l'aéroport Roissy-Charles de Gaulle (CDG), Orly et Le Bourget,
en Mercedes Classe E, V et S.

## Structure

| Fichier | Rôle |
|---|---|
| `index.html` | Page d'accueil (hero, services, flotte, tarifs, réservation, FAQ) |
| `chauffeur-prive-classe-e.html` | Page Mercedes Classe E (formulaire multi-étapes) |
| `chauffeur-prive-classe-s.html` | Page Mercedes Classe S |
| `chauffeur-prive-classe-v.html` | Page Mercedes Classe V |
| `mentions-legales.html` | Mentions légales (RGPD, SIRET) |
| `css/style.css` | Feuille de style (thème luxe noir & or) |
| `js/main.js` | Menu, animations, formulaires |
| `sitemap.xml` / `robots.txt` | SEO |
| `icones/` | Favicons (Android, Apple, MS) |

## Technique

Site statique HTML/CSS/JS, sans dépendance ni build.
SEO : données structurées JSON-LD (TaxiService, FAQ, ItemList, Breadcrumb),
sitemap avec balises image, Open Graph, balises `title` optimisées.
